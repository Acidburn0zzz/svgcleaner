Set number list separator
-------------------------

Set separator for attributes with number list values.
Like ``stroke-dasharray`` or ``points``.

Possible values:

- space
- comma
- comma-space

Default: space

.. GEN_TABLE
.. BEFORE
.. <svg>
..   <polygon fill="green" points="10, 10, 10, 30, 30, 30"/>
.. </svg>
.. AFTER
.. <svg>
..   <polygon fill="green" points="10 10 10 30 30 30"/>
.. </svg>
.. END
