XML Indent
----------

Set indent for XML nodes. Values between 0 and 4 indicates number of indent spaces. Value of ``-1`` indicates no indent.

.. GEN_TABLE
.. BEFORE
.. <svg>
..   <g>
..     <circle fill="green" cx="50" cy="50" r="45"/>
..   </g>
.. </svg>
.. AFTER
.. <svg><g><circle fill="green" cx="50" cy="50" r="45"/></g></svg>
.. END
